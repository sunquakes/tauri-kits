## [0.1.0] - 2026-07-21

### Added

- **Tauri 2 Desktop Runtime** - Cross-platform desktop application foundation for Windows, macOS, and Linux
  - Built-in window management and native APIs
  - Tauri 2 capabilities configuration system

- **React 19 + TypeScript + Vite 7** - Modern frontend development stack
  - Fast hot module replacement (HMR)
  - Type-safe development with TypeScript 5.8
  - Vite 7 optimized build pipeline

- **Ant Design 6** - Enterprise-grade UI component library
  - Responsive layout with Layout, Header, Sider, and Footer components
  - Table, Form, Card, Modal, Drawer, and Tag components
  - Built-in theme customization

- **SQLite3 with sqlx** - Local persistent storage
  - Async Rust driver for SQLite
  - Database initialization on first launch
  - Connection pool management

- **User Management CRUD** - Complete user module
  - Login with username/password authentication
  - User list with pagination and search
  - Create, update, and delete users
  - Default admin account (`admin` / `admin123`)

- **ECharts Dashboard** - Data visualization examples
  - Bar chart with mock data
  - Line chart with mock data
  - Pie chart with mock data

- **i18n Internationalization** - English and Chinese support
  - Built-in language switching
  - Full translation coverage for UI labels
  - i18next + react-i18next integration

- **React Router 7** - Client-side routing
  - Route guards with authentication protection
  - Nested layout routes
  - Login redirect and navigation

- **CLI Scaffolding Tool** - `tkits` command-line utility
  - Interactive project initialization
  - Template-based project creation
  - Global installation support

- **Project Documentation**
  - English and Chinese README files
  - AGENTS guide for AI coding agents
  - CHANGELOG standards
